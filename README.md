# HealthyPlus Europe

An immersive English-language corporate website for HealthyPlus, presenting
six focused skincare, haircare, and bodycare formulas.

## Deployment

This repository is a static website and can be deployed directly on Vercel.
No build command or environment variables are required.

## Local preview

Serve this directory with any static web server and open `index.html`.
